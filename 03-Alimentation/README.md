# Alimentation

## Eteindre le système

* Méthode n°1 :

```shell
sudo halt
```

* Méthode n°2 :

```shell
sudo shutdown -h now
```

## Redémarrer le système:

* Méthode n°1 :

```shell
sudo reboot
```

* Méthode n°2 :

```shell
sudo shutdown -r now
```

## Effectuer une pause système

```shell
sleep [duree][unite]
sleep 1s
```
